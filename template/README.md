# Hello widget

This template should help get you started developing with Desktop Widget.

## Project Setup

### ⚙️ Install dependencies
```sh
npm install
```

### 🛠️ Start development server

```sh
npm run dev
```

### 🌐Build online package

```sh
npm run build
```

### 📦 Build offline zip package

```sh
npm run build:offline
```

### ⬆️ Update widget-js dependencies

```shell
npm run update:widgetjs
```
